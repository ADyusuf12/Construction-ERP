## Getting Started

### Accessing the HR Module

1. Log in to Earmark Systems
2. Click on the "HR" menu in the navigation bar
3. You'll be taken to the Employee Directory

### Understanding Your Role

Your access level determines what you can do:

- **Employee**: View your own profile, submit leave requests
- **Manager**: View your team, approve/reject leave requests
- **HR Administrator**: Full access to all employee records
- **System Administrator**: All HR functions plus system configuration

## Employee Directory

### Viewing All Employees

1. Navigate to **HR → Employees**
2. You'll see a list of all employees with:
   - Staff ID
   - Department
   - Position Title
   - Deployment Date
   - Status (Active/On Leave/Terminated)

### Search and Filter

- Use the search bar to find employees by Staff ID or Designation
- Filter by employment status
- Click on any employee to view their detailed profile

### Employee Status Indicators

- **Active**: Green dot - Currently working
- **On Leave**: Yellow dot - Temporarily absent
- **Terminated**: Red dot - No longer employed

## Managing Your Profile

### Viewing Your Profile

1. Go to **HR → Employees**
2. Find your name in the list (or use search)
3. Click on your name to view your profile

### Profile Information

Your profile shows:

- **Personal Information**: Name, Staff ID, Department, Position
- **Employment Status**: Current status and dates
- **Performance Score**: Your performance rating
- **Leave Balance**: Available leave days
- **Contact Information**: Email and other details

### Updating Personal Information

1. Go to your profile page
2. Click the "Update Personnel File" button
3. Make changes to:
   - Department
   - Position Title
   - Hire Date
   - Status (if you have permission)
   - Leave Balance
   - Performance Score
4. Click "Save Changes"

### Adding Personal Details

If your personal details aren't complete:

1. Go to your profile page
2. Click "Add Personal Details" (if available)
3. Enter:
   - First Name and Last Name
   - Date of Birth
   - Gender
   - Marital Status
   - Contact Information
   - Banking Details
   - Identification Information
4. Click "Save Details"

## Leave Requests

### Submitting a Leave Request

1. Go to **HR → Leave Requests**
2. Click "Initiate Request"
3. Fill in the form:
   - **Start Date**: When your leave begins
   - **End Date**: When your leave ends
   - **Reason**: Why you're taking leave
4. Click "Submit Request"

### Leave Request Status

- **Pending**: Waiting for manager approval
- **Approved**: Manager has approved your leave
- **Rejected**: Manager has denied your leave
- **Cancelled**: Leave request was cancelled

### Viewing Your Leave Requests

1. Go to **HR → Leave Requests**
2. See a list of all your leave requests
3. Check the status of each request
4. View details of approved/rejected requests

### Leave Balance

- Your available leave days are shown in your profile
- Each approved leave day reduces your balance
- Contact HR if you believe your balance is incorrect

## Manager Functions

### Viewing Your Team

1. Go to **HR → Employees**
2. Look for employees with you as their manager
3. Or search for "My Team" if available

### Approving Leave Requests

1. Go to **HR → Leave Requests**
2. Find pending requests from your team members
3. Click on a request to view details
4. Choose to:
   - **Approve**: Click the green checkmark button
   - **Reject**: Click the red X button and confirm
5. The employee will be notified of your decision

### Viewing Team Performance

1. Go to **HR → Employees**
2. Look for performance scores in team member profiles
3. Use this information for performance reviews

### Managing Team Information

1. Go to a team member's profile
2. Click "Update Personnel File"
3. Make necessary changes
4. Save your changes

## HR Administrator Functions

### Creating New Employees

1. Go to **HR → Employees**
2. Click "Add Resource" (green button)
3. Fill in the form:
   - **Staff ID**: Unique employee number
   - **Department**: Department name
   - **Position Title**: Job title
   - **Deployment Date**: Start date
   - **Status**: Active/On Leave/Terminated
   - **Leave Balance**: Annual leave days
   - **Performance Score**: Rating (optional)
   - **Manager**: Select from existing employees
   - **User Account**: Link to system user (optional)
4. Click "Save Employee"

### Managing Employee Records

1. Go to **HR → Employees**
2. Click on any employee to view details
3. Use the tabs to navigate between:
   - **Employment**: Basic employment info
   - **Personal Details**: Name, contact, banking info
   - **Financials/Payroll**: Salary information (if permitted)
   - **History**: Employment history
   - **Documents**: Attached documents

### Managing Leave Requests

1. Go to **HR → Leave Requests**
2. View all leave requests in the system
3. Approve or reject requests as needed
4. Monitor leave balances across the organization

### Organizational Structure

1. Use the manager field when creating/editing employees
2. Build reporting hierarchies
3. View organizational structure through employee relationships

## Common Tasks

### Onboarding a New Employee

**For HR Administrator:**

1. Create employee record with basic information
2. Add personal details (name, contact, banking)
3. Link to user account if system access needed
4. Set initial leave balance
5. Assign to department and manager

**For Manager:**

1. Review new team member's profile
2. Set performance expectations
3. Ensure leave balance is correct
4. Welcome the new employee

### Processing Leave Requests

**For Employee:**

1. Plan leave in advance
2. Submit request with accurate dates
3. Provide clear reason for leave
4. Check status regularly
5. Follow up if not approved within reasonable time

**For Manager:**

1. Review requests promptly
2. Consider team coverage when approving
3. Provide clear feedback if rejecting
4. Update team on approved leave dates
5. Ensure smooth handover of responsibilities

### Updating Employee Information

**When to Update:**

- Department changes
- Position promotions
- Status changes (e.g., returning from leave)
- Personal information updates
- Performance score changes

**How to Update:**

1. Go to employee profile
2. Click "Update Personnel File"
3. Make necessary changes
4. Save changes
5. Notify affected parties if needed

## Troubleshooting

### Cannot Find Employee

- Check spelling in search
- Try partial name search
- Verify employee is active (not terminated)
- Contact HR if still not found

### Leave Request Not Showing

- Check if you're logged in as the correct user
- Verify request was actually submitted
- Check if request is pending approval
- Contact manager if status hasn't changed

### Cannot Approve Leave

- Verify you're the manager of the employee
- Check if you have permission to approve
- Ensure request is in "Pending" status
- Contact HR if you should have approval rights

### Personal Details Missing

- Check if details were saved correctly
- Verify you have permission to view details
- Contact HR to add missing information
- Ensure all required fields are filled

### Performance Score Not Updating

- Verify you have permission to update scores
- Check if score field is enabled
- Contact HR for performance review process
- Ensure score is within valid range

### Cannot Create Employee

- Verify you have HR administrator rights
- Check if Staff ID is unique
- Ensure all required fields are filled
- Contact system administrator if issue persists

## Tips and Best Practices

### For Employees

- Keep your personal information up to date
- Plan leave well in advance
- Provide clear reasons for leave requests
- Check your leave balance regularly
- Communicate with your manager about planned absences

### For Managers

- Review leave requests within 2 business days
- Consider team workload when approving leave
- Keep team informed about approved leave
- Document performance discussions
- Maintain accurate team records

### For HR Administrators

- Verify Staff IDs are unique before creating
- Keep personal details confidential
- Process status changes promptly
- Maintain accurate organizational structure
- Regular audits of employee records
