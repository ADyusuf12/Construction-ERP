## Getting Started

### Accessing Accounting Information

1. Log in to Earmark Systems
2. Click on the "Accounting" menu in the navigation bar
3. Select "Payroll Management" to view all salary records
4. Or select "Financial Ledger" to view all transactions

### Understanding Your Role

Your access level determines what you can do:

- **Employee**: View your own salary records and payslips
- **Manager**: View team member salaries (if permitted)
- **Accountant**: Full access to create, edit, and process financial records
- **System Administrator**: All accounting functions plus system configuration

## Financial Transactions

### Viewing Financial Transactions

1. Go to **Accounting → Financial Ledger**
2. See a list of all financial transactions
3. Each transaction shows:
   - Date
   - Description
   - Type (Payment/Receipt)
   - Amount
   - Status (Pending/Paid/Failed)

### Transaction Types

- **Payment**: Money going out of the organization
- **Receipt**: Money coming into the organization

### Transaction Status

- **Pending**: Transaction created but not processed
- **Paid**: Transaction has been completed
- **Failed**: Transaction could not be processed

## Salary Management

### Viewing Salary Records

1. Go to **Accounting → Payroll Management**
2. See a list of all salary records
3. Each salary shows:
   - Employee Name
   - Base Pay
   - Allowances
   - Deductions
   - Net Pay
   - Status (Pending/Processed/Paid)

### Salary Status

- **Pending**: Salary not yet processed
- **Processed**: Salary calculated but not paid
- **Paid**: Salary has been disbursed

## Employee Functions

### Accessing Your Payslip

1. Go to **Accounting → Payroll Management**
2. Find your name in the salary list (or use search)
3. Click on your salary record to view details

### Understanding Your Payslip

Your payslip shows:

- **Employee Information**: Name and Staff ID
- **Base Salary**: Your fixed monthly salary
- **Allowances**: Additional payments (bonuses, overtime, etc.)
- **Deductions**: All deductions itemized
- **Net Pay**: Final amount you receive
- **Status**: Payment status (Pending/Processed/Paid)

### Salary Breakdown

- **Base Pay**: Your fixed monthly salary amount
- **Allowances**: Additional compensation added to your pay
- **Deductions**: Amounts subtracted from your pay
- **Net Pay**: What you actually receive (Base + Allowances - Deductions)

### Common Questions

**Q: Why is my net pay different from my base salary?**
A: Your net pay is calculated as: Base Pay + Allowances - Deductions. Deductions may include taxes, pension contributions, insurance, or other items.

**Q: When will I be paid?**
A: Check the status of your salary. If it's "Paid," the payment has been processed. If it's "Pending" or "Processed," it's still being processed.

**Q: Can I see my salary history?**
A: Yes, all your past salary records are available in the Payroll Management section.

## Accountant Functions

### Creating Financial Transactions

1. Go to **Accounting → Financial Ledger**
2. Click "Post Transaction" (green button)
3. Enter transaction details:
   - **Date**: When the transaction occurred
   - **Description**: What the transaction is for
   - **Amount**: The monetary value
   - **Type**: Payment (outgoing) or Receipt (incoming)
   - **Reference**: Optional reference number
   - **Notes**: Additional context
4. Click "Create Transaction"

### Marking Transactions as Paid

1. Go to **Accounting → Financial Ledger**
2. Find the transaction you want to mark as paid
3. Click "Mark as Paid" button
4. Transaction status changes to "Paid"

### Creating Salary Batches

1. Go to **Accounting → Payroll Management**
2. Click "New Salary Batch"
3. Enter:
   - **Batch Name**: Descriptive name (e.g., "January 2026 Payroll")
   - **Period Start**: First day of the pay period
   - **Period End**: Last day of the pay period
4. Click "Create Batch"

### Adding Salaries to a Batch

1. Go to the batch you created
2. Click "Add Salary"
3. Select an employee from the dropdown
4. Enter:
   - **Base Pay**: Employee's fixed salary
   - **Allowances**: Any additional payments
5. Click "Save Salary"

### Adding Deductions to a Salary

1. Go to the salary record
2. Click "Add Deduction"
3. Select deduction type:
   - **Tax**: Income tax
   - **Pension**: Retirement contributions
   - **Insurance**: Health or other insurance
   - **Other**: Any other deduction
4. Enter the amount
5. Add notes (optional)
6. Click "Save Deduction"

### Processing Salary Batches

1. Review all salaries in the batch
2. Verify all information is correct
3. Click "Mark as Processed"
4. Batch status changes to "Processed"

### Marking Salary Batches as Paid

1. After processing, verify all salaries
2. Click "Mark as Paid"
3. Batch status changes to "Paid"
4. All salaries in the batch are marked as paid

## Common Tasks

### Monthly Payroll Processing (Accountant)

**Step 1: Create Batch**

- Create new batch for the month
- Set correct period dates

**Step 2: Add Employees**

- Add all active employees to the batch
- Enter correct base pay from employment records

**Step 3: Add Allowances**

- Include any bonuses or overtime
- Verify allowance amounts

**Step 4: Add Deductions**

- Add tax deductions
- Include pension contributions
- Add insurance deductions
- Include any other applicable deductions

**Step 5: Review and Process**

- Review all calculations
- Verify net pay amounts
- Mark batch as processed

**Step 6: Final Payment**

- Perform final verification
- Mark batch as paid
- Generate payroll report

### Recording a Financial Transaction (Accountant)

**Step 1: Determine Transaction Type**

- Is it money coming in (Receipt) or going out (Payment)?

**Step 2: Gather Information**

- Date of transaction
- Description of what it's for
- Amount
- Any reference numbers
- Additional notes

**Step 3: Create Transaction**

- Go to Financial Ledger
- Click "Post Transaction"
- Enter all required information
- Click "Create Transaction"

**Step 4: Mark as Paid**

- Once verified, mark transaction as paid
- Update status to "Paid"

### Checking Your Payslip (Employee)

1. Go to Payroll Management
2. Find your salary record
3. Review the breakdown:
   - Base pay
   - Allowances
   - Deductions
   - Net pay
4. Verify the information is correct
5. Contact HR if you have questions

### Reviewing Team Salaries (Manager)

1. Go to Payroll Management
2. Search for team members
3. View their salary records
4. Check payment status
5. Use information for performance reviews

### Reconciling Financial Transactions (Accountant)

1. Go to Financial Ledger
2. Filter by date range
3. Verify each transaction against source documents
4. Update statuses as confirmed
5. Generate reconciliation report

## Troubleshooting

### Cannot Find My Salary

- Check if you're logged in as the correct user
- Verify your employment status is active
- Check if salary batch has been created for the current period
- Contact HR if salary is missing

### Net Pay Amount Seems Wrong

- Review all deductions listed
- Check if allowances are included correctly
- Verify base pay amount
- Contact HR for clarification on specific deductions

### Cannot Add Deduction

- Verify you have accountant or admin role
- Check if salary is in correct status
- Ensure deduction amount is positive
- Contact system administrator if issue persists

### Transaction Not Showing

- Check if transaction has been created
- Verify you have permission to view transactions
- Check date filters if applied
- Contact accountant if transaction is missing

### Cannot Mark Transaction as Paid

- Verify transaction is in "Pending" status
- Check if you have permission to mark as paid
- Ensure transaction details are complete
- Review transaction for any errors

### Salary Batch Not Showing

- Check if batch has been created for the current period
- Verify you have permission to view batches
- Check date filters if applied
- Contact accountant if batch is missing

### Deduction Not Appearing

- Check if deduction was saved correctly
- Verify deduction type is valid
- Ensure amount is greater than zero
- Check if salary status allows modifications

## Tips and Best Practices

### For Accountants

- Always verify amounts before marking as paid
- Use batch processing for efficiency
- Keep detailed notes for audit trails
- Regular reconciliation with financial records
- Process salaries before payment deadlines

### For Employees

- Review your payslip each month
- Keep personal information updated
- Understand your deduction types
- Contact HR for any discrepancies
- Save your payslips for records

### For Managers

- Review team salaries regularly
- Understand salary trends in your team
- Use salary information for performance discussions
- Respect salary confidentiality
- Report any discrepancies to HR
